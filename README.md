## 📖 About the Project

This project is a responsive cryptocurrency landing page developed using React. It features a modern user interface and displays cryptocurrency information using the CoinGecko API.

---

## ✨ Features

- Responsive Design
- Modern UI
- Hero Section
- Featured Cryptocurrency Section
- Signup Section
- Footer
- Live Cryptocurrency Data
- Fast and Clean Layout

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6)
- CSS3
- Axios
- CoinGecko API

---

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/zahrabedi82/react-project.git
```

Go to the project directory

```bash
cd react-project
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm start
```
